Deleting

Users can delete the data used by the extension. This action does not change any data on YouTube. Description of steps to delete data:

  1. Turn off the browser extension
  2. Restart the computer with the installed extension turned off.
  3. Open your browser after restarting and make sure that the extension is turned off, if it is not turned off, start from step 1.
  4. Open Google Drive on the web and find the av-soft folder there. If you have not found this folder, then you need to look for it in all of your accounts with which the Google Drive works.
  5. Delete the found folder av-soft.
  6. Go to your Google Drive trash and find av-soft folder there.
  7. Empty the Google Drive trash where you see the found av-soft folder.

With respect to any extension provided by playlistshistory.com, you agree:

  1. to delete data every 30 days using the above steps;
  2. to delete data after each revocation of access using the above steps;
  3. do not store data generated by any playlistshistory.com service for more than 30 days;
  4. do not copy data from av-soft folder anywhere;
  5. do not sync the av-soft folder data with anything;
  6. do not make a back up copy of the av-soft folder;
  7. do not share the av-soft folder with anyone;
  8. do not create the av-soft folder outside of the extension;
  9. all data in the av-soft folder is controlled by the extension;
